# ECG_Filters

###General Usage:
- To compile:
```
-$ chmod +x build.sh
-$ chmod +x run.sh 
```

- To Run:
```
-$ ./build.sh
-$ ./run.sh <inputECGFileName> <outputFileName>
```
###Things needed
- gcc compiler to compile and run the C files
- Linux (we used Ubuntu) operating system
- Octave 4.0 using the "signal" package
- pdflatex to compile the tex file
